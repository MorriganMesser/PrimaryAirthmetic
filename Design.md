#Project Structure

## Web Structure
登录页面=>注册页面:点击注册
注册页面=>登录页面:注册成功
注册页面=>弹出错误:注册失败
登录页面=>主页:点击登录
登录页面=>弹出错误:登录失败
主页=>考试页面:选择题库，点击考试
考试页面=>结果页面:点击提交
主页=>记录页面:点击查看记录
## Function Structure
数字类NumberClass
表达式生成&&表达式计算=>题库生成&&答案生成=>结果生成&&记录生成

# 代码规范

## 命名规范
1. JAVA文件：每个单词首字母大写
2. 网页文件：全部小写
3. 方法名：第一个单词首字母小写之后首字母大写
4. 类名：每个单词首字母大写
5. 变量名：第一个单词首字母小写之后首字母大写
